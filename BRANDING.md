# Branding Guidelines — EWM App

**v1 uses generic placeholder branding.** Swap in real EWM assets once they're provided. Do not copy MLS / Purple Admin.

## Core

- **Company name**: Elevated Workforce Management
- **Short name**: EWM
- **Tagline**: *People. Performance. Elevated.*
- **Mission**: *We Build Workforce Programs That Last*
- **Website**: elevated-workforce.com

## Logo (v1 placeholder)

- Use a simple text wordmark "EWM" in the primary color
- Generate a minimal app icon from the wordmark for dev builds
- **Deferred** — obtain proper SVG/PNG pack from EWM before any real pilot or store submission

## Color palette (v1 placeholder)

- Primary: `#1E3A8A` (navy) — fits "elevated" positioning
- Accent: `#0EA5E9` (sky) — for CTAs, active states
- Success: `#16A34A`
- Warning: `#D97706`
- Error: `#DC2626`
- Neutral: Tailwind slate scale

All colors defined as Tailwind theme tokens so swapping to real EWM brand later is a one-file change.

## Typography (v1 placeholder)

- Web: Inter
- iOS: SF Pro (system default)
- Android: Roboto (system default)

## App-specific

- **App name (stores)**: "EWM Workforce" or "EWM Productivity" — confirm with EWM
- **App icon**: EWM mark on brand primary, clean and legible at 1024×1024
- **Splash**: wordmark + tagline
- **Admin portal title**: "EWM Admin" (parallel to MLS's "Purple Admin")
- **Loading/empty states**: use tagline fragments ("Elevating the team...")

## Voice & tone

From EWM's site copy:
- Operator-built, hands-on, no-fluff
- Emphasis on *accountability*, *performance*, *compliance*
- Speaks to operations people, not HR people
- Short declarative sentences

Apply in:
- Push notification copy
- Empty states
- Error messages
- Onboarding walkthroughs

## What NOT to do

- Do not reuse MLS / Purple Admin purple color scheme
- Do not copy MLS's UI verbatim — EWM is a competitor product
- Do not reference "MLS," "Purple Admin," or "Team Safe Plus" anywhere user-facing
